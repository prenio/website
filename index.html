Predicting YouTube Video Clickbait Potential and Predicting Metadata Statistics


With YouTube being the largest social media platform and one of the most visited websites in the world, it is only natural that clickbait stays present and ungoverned. This project aimed to determine if it is possible to detect clickbait in a video based on primarily the title of the video and other metadata numbers. Using a sentiment analysis and the Naive Bayes Classification method, clickbait potential models were able to be created. The report can be viewed here.

MLB Linear Regression


Baseball is the textbook statistics sport. It's no surprise here that it was the focus for at least one stats project. The goal was to find the 2 most influential dependent variables on the independent variable: number of wins. Arguably the most important variable in the entire game. Using SAS and linear regression techniques, a model was able to be created with the 2 most important variables to determine the number of wins. The report can be viewed here.
